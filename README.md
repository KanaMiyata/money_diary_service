# このプロジェクトについて
このプロジェクトでは、日にち、カテゴリー、金額、コメントを管理するREST APIを実装します。



| 列名          | データ型       | 説明                                         |
|-------------|------------|--------------------------------------------|
| id          | INTEGER    | レコードの一意の識別子                                |
| DATE 　　　　   | DATE       | 収支が発生した日時                                  |
| TRANSACTION_TYPE | VARCHAR    | 収入か、支出かを示す                                 |
| ITEM_NAME   | VARCHAR    | 収入や支出の種類を示す                                |
| AMOUNT      | DECIMAL    | 金額（通貨の小数点以下の桁数を考慮）                         |
| COMMENT　　　　　　　 | TEXT　　　　　　　 | その他の情報やコメント                                |
